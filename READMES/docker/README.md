### o que é docker e containers?

- "É uma Engine de Administração de Containers."
- "Docker é um conjunto de produtos de plataforma como serviço que usam virtualização de nível de sistema operacional para entregar software em pacotes chamados contêineres. Os contêineres são isolados uns dos outros e agrupam seus próprios softwares, bibliotecas e arquivos de configuração."
- Todos containers que estão rodando em uma maquina especifica irão utilizar aquele mesmo kernel.
- O container utiliza o kernel da maquina host.
- Tem uma necessidade de processamento muito menor do que uma maquina virtual teria.
- Os containers só funcionam em sistemas operacionais baseados em linux.
- Com containers nós podemos empacotas softwares com varios niveis de isolamento. Controlando a quantia de memoria, rede, cpu, entre outro, que o container ira utilizar.
